## bash/vim dotfiles

To update, `cd` into your local `dotfiles` repository and then:

* for bash

  ```bash
  source bash.sh
  ```

* for vim

  ```bash
  source vim.sh
  ```

Alternatively, to update while avoiding the confirmation prompt:

* for dash

  ```bash
  set -- -f; source bash.sh
  ```
* for vim

  ```bash
  set -- -f; source vim.sh
  ```
